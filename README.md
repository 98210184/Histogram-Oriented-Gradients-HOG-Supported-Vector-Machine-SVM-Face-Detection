# HOG + SVM for detect Human Faces Using Non-Maximum-Suppresion and HeatMap

In this project, we take the advantage of SK-Learn platform to extract features from the Stanford Dataset using Histogram of Oriented Gradients (HOG). Then, we make the most of Supported Vector Machine (SVM), as a supervised learning method, to map the features to the assigned labels. After extracting the SVM model, we use Non-Maximum Suppression and Heatmap methods to find the best bounding box on the face in a given image.
